# ALBERLICSAC Landing Page

Landing page en Next.js, React y Tailwind CSS para servicios de mudanza,
transporte, carga, embalaje, plataforma hidráulica, traslado de mascotas,
traslado de vehículos y limpieza en Perú.

## Desarrollo

```bash
bun install
bun run dev
```

Luego abre `http://localhost:3000`.

Para generar la versión estática:

```bash
bun run build
bun run start
```

Los datos principales de la empresa, WhatsApp y dominio SEO se editan en
`app/data.ts`.
